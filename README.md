# 灵感笔记（IdeaVault）

面向 Android 12 及以上版本的隐私记事本。适合记录灵感、临时信息和需要额外保护的账号资料。应用名称跟随系统语言：中文显示“灵感笔记”，英文显示“IdeaVault”。

## 功能

- 新建、编辑、搜索、置顶和删除笔记
- 敏感笔记隐藏正文预览
- 生物识别或设备锁屏凭据解锁
- 本地笔记使用 Android Keystore AES-256-GCM 加密
- 禁止系统备份、截图和最近任务预览
- 自托管多用户同步，用户可在 App 内自行注册和设置密码
- 每个账户的密码摘要、会话、密钥和笔记相互隔离
- 新增、修改、置顶或删除后自动同步，启动时自动拉取
- 右上角显示用户名；绿色勾表示服务器正常且最近一次同步完成
- 30 天可配置登录会话、显式登出、修改密码后撤销旧会话

> 它不是经过审计的专业密码管理器。长期保存重要密码仍建议使用成熟的密码管理器。

## 构建

安装 Android SDK 35 和 JDK 17，使用 Android Studio 打开项目，或运行：

```text
./gradlew assembleDebug
```

最低 Android 12（API 31），目标 SDK 35。GitHub Actions 会测试 Go 服务端、构建 Docker 镜像、校验 Compose、构建签名 APK，并在 `v*` 标签上发布 Release。

## 自托管同步

2.4.0 支持用户直接在 App 中输入服务器地址、用户名和密码完成注册或登录。密码通过 Argon2id 加盐摘要保存，客户端登录后只保存由服务器签发的限时随机会话令牌，并使用 Android Keystore 加密保护该令牌。

服务端仍管理每位用户的数据加密密钥，因此拥有 VPS 或数据卷管理权限的人理论上能取得密钥；这不是零知识同步。完整 Nginx 部署、升级、关闭公开注册及 owner 密码迁移步骤见 [`server/README.md`](server/README.md)。
